## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/kratin01/Nike-Landing-Page.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open the app in your browser:

   ```bash
   http://localhost:3000
   ```
